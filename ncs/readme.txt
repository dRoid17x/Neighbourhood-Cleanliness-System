How to run the Neighbourhood Cleanliness System (NCS) Project
1. Download the  zip file

2. Extract the file and copy ncs folder

3.Paste inside root directory(for xampp xampp/htdocs, for wamp wamp/www, for lamp var/www/html)

4. Open PHPMyAdmin (http://localhost/phpmyadmin)

5. Create a database with name ncs

6. Import ncs.sql file(given inside the zip package in SQL file folder)

7. Run the script http://localhost/ncs (frontend)

8. For admin panel

http://localhost/ncs/admin  (admin panel)
